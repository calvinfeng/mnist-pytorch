# MNIST: Hello World of Machine Learning

## Setup

- Python 3.8
- PyTorch
- Numpy
- Matplotlib
- Jupyter

```sh
python -m venv env
source env/bin/activate
(env) pip install -r requirements.txt
```

## Run

Launch Jupyter notebook

```sh
(env) jupyter notebook
```
